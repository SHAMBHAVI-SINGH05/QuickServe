# ⚡ QuickServe – AI Powered Canteen Queue Optimizer

QuickServe is an intelligent canteen management system designed to reduce waiting time and improve food service efficiency using machine learning and real-time analytics.

The platform predicts queue wait times, identifies peak rush hours, and highlights popular food items to help students and canteen staff make faster and smarter decisions.

---

## 🚀 Features

- ⏱ AI-Based Wait Time Prediction
- 📈 Peak Hour & Rush Analysis
- 🍔 Popular Food Item Insights
- 📊 Interactive Dashboard Interface
- 🔐 User Authentication System
- ⚡ Fast Backend API Integration
- 🧠 Machine Learning Prediction Models
- 💻 Responsive User Interface

## ⚙️ Installation & Setup

### Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
